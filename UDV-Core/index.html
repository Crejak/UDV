UDV Examples

UDV is a JavaScript client based on iTowns allowing to visualize, analyse and interact with urban data. It is developed as part of the Virtual City Project.

This page lists the demos and examples of UDV.

Demos

List of demonstrations combining modules and / or extensions of UDV.

  • UDV Demo

    Demonstrate the modules Temporal and ConsultDoc.

    In this demo you can navigate in a 4D (3D + time) representation of the city. You can also consult documents in a document browser and search them using keywords and other fields.

  • UDV Authentication and Validation Demo

    Demonstrate the modules Authentication and Validation.

    This Demo is an improvement of the above. You can create user accounts and authenticate. The documents are linked to the user who created them and the submission of a new document must be checked and validated by a moderator.

  • Augmented Model of Mediation

    Demonstrate the Mediation Tool.

    In this demo you can consult several map of the city at once. Also you can interact with layers so you can add extra data, and create an customize model with the essential information.

Individual Modules and Extensions Demos

Each individual module and extension should have an associated example. They are listed here.

  • Consult Documentation

    Demonstrate the ConsultDoc Module.

    In this demo you can consult documents in a document browser and search them using keywords and other fields.

  • Guided Tour

    Demonstrate the GuidedTour Module.

    In this demo you can try the mode guided tour which is a virtual navigation in a 4D (3D + time) and documented representation of the city.

  • Contribute

    Demonstrate the Contribute Extension.

    In this demo you can try the Contribute extension which allows to create, update and delete documents. You can also consult documents in a document browser and search them using keywords and other fields.
